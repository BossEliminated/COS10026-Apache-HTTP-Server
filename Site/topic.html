Apache HTTP Info

What is apache?

Apache is an open source server application, it receives HTTP directory requests from users and sends them the information at that address. It can send files or handle reverse proxy requests.

Background

Apache is non-copyleft open source and owned by the Apache Software Foundation. They are a nonprofit organisation and based in America. The group was established on March 25, 1999 today continuing to support a number of open source software projects.

Features

  1. Storage & Bandwidth
  2. Root SSH Access
  3. Unlimited Domain Hosting
  4. PHP5 Support
  5. SSL Secure Server
  6. IPv6 Support
  7. Canned CGI Scripts
  8. Full CGI Access
  9. Network Isolation
